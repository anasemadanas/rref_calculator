Linear Algebra

Linear Algebra is a branch of mathematics that studies vectors, matrices, and systems of linear equations.

Matrix

A matrix is a rectangular arrangement of numbers organized into rows and columns. 

[ 1  2  3 ]
[ 4  5  6 ]

Systems of Linear Equations

A system of equations can be represented using an augmented matrix. 

x₁ + 2x₂ = 5
3x₁ + 4x₂ = 7


[ 1  2 | 5 ]
[ 3  4 | 7 ]

Row Operations

There are three elementary row operations:

  • Swap two rows:
    R₁ ↔ R₂
  • Multiply a row by a constant:
    R₁ → kR₁
  • Add or subtract rows:
    R₂ → R₂ + kR₁

These operations preserve the solution of the linear system.

Types of Solutions

A system of linear equations can have different types of solutions depending on the relationship between equations and variables.

1. One Unique Solution (Consistent System)

A system has exactly one solution when every variable has a pivot position. The number of pivots is equal to the number of variables. 

[ 1  0 | 3 ]
[ 0  1 | 2 ]

Solution: 

x₁ = 3
x₂ = 2

2. Infinite Solutions (Consistent System)

A system has infinitely many solutions when there are free variables. This happens when there are fewer pivots than variables and no contradiction exists. 

[ 1  2 | 5 ]
[ 0  0 | 0 ]

The second row represents: 

0x₁ + 0x₂ = 0

which is always true, so the system has infinite solutions.

3. No Solution (Inconsistent System)

A system has no solution when it contains a contradiction. In RREF, this appears as a row where all coefficients are zero but the constant is not zero. 

[ 1  2 | 5 ]
[ 0  0 | 3 ]

The second row represents: 

0x₁ + 0x₂ = 3

This is impossible, so the system is inconsistent.

Consistent vs Inconsistent Systems

System Type Meaning Number of Solutions
Consistent The equations have at least one solution One solution or Infinite solutions
Inconsistent The equations contradict each other No solution
