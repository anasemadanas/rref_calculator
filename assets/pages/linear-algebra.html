Linear Algebra Concepts

Linear Algebra is a branch of mathematics that studies vectors, matrices, and systems of linear equations. It provides the language used to describe many problems in science, engineering, and data analysis.

Matrices

A matrix is a rectangular arrangement of numbers organized into rows and columns. 

[ 1  2  3 ]
[ 4  5  6 ]

Systems of Linear Equations

A system of equations can be represented using an augmented matrix, which places the constants in the last column. 

x₁ + 2x₂ = 5
3x₁ + 4x₂ = 7


[ 1  2 | 5 ]
[ 3  4 | 7 ]

Row Operations

There are three elementary row operations used in Gaussian Elimination:

  • Swap two rows:
    R₁ ↔ R₂
  • Multiply a row by a non-zero constant:
    R₁ → kR₁
  • Add or subtract a multiple of one row from another:
    R₂ → R₂ + kR₁

These operations preserve the solution set of the linear system.

Types of Solutions

A system of linear equations can have different types of solutions depending on the relationship between equations and variables.

1. Unique Solution

A system has exactly one solution when every variable corresponds to a pivot column and no contradiction appears. 

[ 1  0 | 3 ]
[ 0  1 | 2 ]

Solution: 

x₁ = 3
x₂ = 2

2. Infinite Solutions

A system has infinitely many solutions when there are free variables and no contradiction appears. 

[ 1  2 | 5 ]
[ 0  0 | 0 ]

The zero row represents a redundant equation: 

0x₁ + 0x₂ = 0

which is always true, so the system has infinitely many solutions.

3. No Solution

A system has no solution when it contains a contradiction. In RREF, this appears as a row where all coefficients are zero but the constant term is not zero. 

[ 1  2 | 5 ]
[ 0  0 | 3 ]

The second row represents a contradiction: 

0x₁ + 0x₂ = 3

This is impossible, so the system is inconsistent.

Consistent vs Inconsistent Systems

System Type Meaning Number of Solutions
Consistent The equations have at least one solution One solution or infinitely many solutions
Inconsistent The equations contradict each other No solution
