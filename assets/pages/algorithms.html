Algorithms Used in the Solver

This project implements matrix reduction algorithms in JavaScript and shows the intermediate steps so students can follow the logic behind each transformation.

Gaussian Elimination (REF)

Gaussian Elimination transforms an augmented matrix into Row Echelon Form by eliminating entries below each pivot.

Steps

  1. Choose the pivot element with partial pivoting.
  2. Swap rows when a better pivot is available in the current column.
  3. Eliminate the values below the pivot.
  4. Move to the next column and repeat the process.

Example

Before:

1 2 3
2 4 6


Operation:

R₂ = R₂ - 2R₁


After:

1 2 3
0 0 0

Gauss-Jordan Elimination (RREF)

RREF extends REF by normalizing each pivot to 1 and eliminating the entries above it, producing a matrix that is easier to read directly.

Example

Before:

1 3
0 1


Operation:

R₁ = R₁ - 3R₂


After:

1 0
0 1

Implementation

  • createMatrix() Creates matrix input fields dynamically.
  • getMatrix() Reads user input and converts it into a JavaScript array.
  • REF() Applies Gaussian Elimination with partial pivoting.
  • RREF() Applies Gauss-Jordan Elimination and removes coefficients above pivots.
  • saveStep() Stores every operation and matrix state for step-by-step display.
