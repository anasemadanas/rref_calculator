Algorithms

This project implements matrix reduction algorithms using JavaScript.

Gaussian Elimination (REF)

Gaussian Elimination converts a matrix into Row Echelon Form.

Steps:

  1. Find the pivot element.
  2. Swap rows if the pivot is zero.
  3. Eliminate values below the pivot.
  4. Move to the next column.

Example

Before:

1 2 3
2 4 6


Operation:

R₂ = R₂ - 2R₁


After:

1 2 3
0 0 0

Gauss-Jordan Elimination (RREF)

RREF extends REF by making every pivot equal to 1 and removing values above pivots.

Example

Before:

1 3
0 1


Operation:

R₁ = R₁ - 3R₂


After:

1 0
0 1

Implementation

  • createMatrix() Creates matrix input fields dynamically.
  • getMatrix() Reads user input and converts it into a JavaScript array.
  • REF() Applies Gaussian Elimination.
  • RREF() Applies Gauss-Jordan Elimination.
  • saveStep() Stores every operation and matrix state.
